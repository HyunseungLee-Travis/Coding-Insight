# front()와 back()
front()는 리스트의 가장 앞을 돌려주고 back()은 가장 뒤입니다.

```
#include <iostream>
using namespace std;

int main()
{
  list<int> a = {1, 2, 3};

  cout << a.front();
  cout << a.back();

  return 0;
}
```

# sort()
sort() 메서드를 사용하면 리스트가 위에서 아래로 정리가 됩니다.

```
#include <iostream>
using namespace std;

int main()
{
  list<int> a = {1, 2, 3};

  cout << a.sort();

  return 0;
}
```
