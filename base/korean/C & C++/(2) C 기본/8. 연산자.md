# 기본 연산자
+, -, *, /를 사용하면 더하기, 빼기, 곱하기, 나누기를 할 수 있습니다.

```
#include <stdio.h>

int main()
{

  printf("%d", 2 + 2);
  printf("%d", 2 - 2);
  printf("%d", 2 * 2);
  printf("%d", 2 / 2);

  return 0;
}
```

# 할당 연산자
할당 연산자란 어떤 숫자에 연산을 한 후 연산한 값을 그 변수 안에 넣는 작업입니다.

원래는 연산을 하면 그 변수는 바뀌지 않지만 할당 연산자는 새로운 값을 그 변수 안에 저장합니다.

예를 들면, a += 2는 원래 a 값에 2를 더하고 그 값을 a 안에 저장합니다.

```
#include <stdio.h>

int main()
{
  int a = 2;

  a += 2;
  a -= 2;
  a *= 2;
  a /= 2;

  printf("%d", a);

  return 0;
}
```

# 기타 연산자
++을 사용하면 1을 증가시키고 --를 사용하면 1이 감소됩니다.

```
#include <stdio.h>

int main()
{
  int a = 2;

  a++;

  printf("%d", a);

  a--;

  printf("%d", a);

  return 0;
}
```
