# 화살표 ->

화살표 키워드는 struct와 union(포인터로)의 아이템을 사용할 수 있게 도와줍니다.

예를 들면, 먼저 간단한 struct 만들겠습니다.

```
struct Books {
   char  title[50];
   char  author[50];
   char  subject[100];
   int   book_id;
};


struct Books book;
```

# -> 사용하기
book의 title은 ->를 사용해서 값을 찾을 수 있습니다.

```
#include <stdio.h>

struct Books {
   char  title[50];
   char  author[50];
   char  subject[100];
   int   book_id;
};


struct Books percyjackson;

int main()
{
  (&percyjackson)->title = "Rick Riordan";
  return 0;
}
```

# ->에 대한 설명
->는 쉽게 말하지만 (*var).var1과 같습니다.

포인터의 '.'입니다.
