# 화살표 ->

화살표 키워드는 struct와 union(포인터로)의 아이템을 사용할 수 있게 도와줍니다.

예를 들면, 먼저 간단한 struct 만들겠습니다.

```
struct student {
    char name[80];
    int age;
    float percentage;
};

struct student* emp;
```

# -> 사용하기

->는 struct 또는 union의 값을 가지고 오는데 사용합니다.<br><br>

'.'와 비슷한데 변수가 포인터일때 사용합니다.<br><br>

```
emp = (struct student*)malloc(sizeof(struct student));

emp->age = 18;

printf("%d", emp->age);
```
