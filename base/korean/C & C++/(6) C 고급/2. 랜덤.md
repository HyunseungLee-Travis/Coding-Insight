# 랜덤
랜덤이라는 것은 무작위로 또는 패턴 없이 행동한다는 뜻입니다.

랜덤 함수는 stdlib.h 라이브러리 안에 있습니다.

```
#include <stdio.h>
#include <stdlib.h>
```

# rand()
rand() 함수는 무작위로 0부터 2147483647(C 언어 int의 가장 큰 값) 사이의 수를 선택합니다.

```
#include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main()
{
    srand(time(0));
    printf("%d", rand());

    return 0;
}
```
