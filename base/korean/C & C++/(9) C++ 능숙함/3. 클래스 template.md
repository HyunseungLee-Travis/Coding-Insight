# 크래스 template이란?
클래스 template은 template이지만 일반적인 변수들을 가질 수 있는 클래스를 뜻합니다.

# 클래스 template 사용하기
```
#include <iostream>
using namespace std;

template<class T>
class A
{
    public:
      T num1 = 5;
      T num2 = 6;

      void add()
      {
          cout << num1+num2
      }
};
```
