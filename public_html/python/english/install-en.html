Python programming language needs to be converted into 0s and 1s for the computer to understand.

This process is called compiling. We first need to install a compiler for Python which converts Python to binary digits.

How to use IDLE

We will now see how we can use the tool in Python called IDLE.

It lets us write Python code in them.



1. To find IDLE app, click the search in the bottom of you screen and search 'IDLE'



2. If you press enter or click open, the following screen will appear



3. We can type Python codes inside. Try print("Hello World").

5. Press enter and you will see the text 'Hello World'
