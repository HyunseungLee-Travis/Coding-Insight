How to download VSCode

1. Go to VSCode Download Page.

2. Choose the correct OS for you.

+ If there is a window button, your system is windows - choose left(otherwise MAC - choose right).

3. When the download is complete, click the samll box.

4. Click on Agree, Next, Install and Finish button.

Prepare Python in VSCode

1. On the left side of the screen where the sidebar floats, there is a four-cube button.

Click it

3. Type in "python" keyword

4. Click the install button of the first box that appears on screen.

5. Type in code runner and install it, too

How to use Python in VSCode

1. We need a folder to store our codes in.

2. CTRL+SHIFT+E or click the file icon on top left.


3. Click the "Open Folder" button.

4. After choosing a folder, you will see 4 small icons. Click the left one.

5. Type in a name with'.py' at the end.

6. Type in python code. print("Hello World")

7. Click the trianlge on the top right corner.


Hello World is printed to the console.

.py?

We have to put '.py' at the end of file to make it a Python file.

This is called an extension where we specify the type of file.
