What is Anonymous Function?

Anonymous function is a type of function that does not have a name.

In another word, anonymous functions are created and used right away while normal functions are created for later use.









Anonymous Function
Computer Science Lab (CSL) 33






Lambda

To declare anonymous functions,

1. type in the 'lambda' keyword
2. insert parameters with commas
3. insert ':'
4. Put in the return value

def runThis(a):
 a(6)

x = runThis(lambda a: 10*a)

print(x) If you have more than 1 parameter, put in ','(Comma) between the parameter.

def runThis(a):
 a(6)

x = runThis(lambda a: 10*a)

print(x)





Difference between anonymous functions and functions

The functions below does the same thing.

x = lambda a,b: a*b

def x(a, b):
 return a*b To use anonymous functions like functions, store lambda inside variable and use it like a function name.

Note that the last thing to put in anonymous function is the return value.
