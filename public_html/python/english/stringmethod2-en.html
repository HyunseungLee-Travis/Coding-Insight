Python Character Trim

I'm going to make a program to filter out bad words.

badWord = input("Bad word: ")
badSentence = ("I am not " + badWord + ".")
filterBadWord = (badSentence.partition(badWord))

print(filterBadWord[0] + "***" + filterBadWord[2]) First, I asked a bad word and made a sentence that means 'I'm not a {bad word}'. Then, using the partition() method, I cut the sentence based on the bad word and put '**' instead of the bad word.

Let's take another example:

partitionedSentence = ("Hello. My name is Hyunseung.".partition("name"))

print(partitionedSentence) In this example, truncated by name. If you print partitionedSentence, you can observe that ("Hello. My ", "name", " is Hyunseung") is output to the terminal.

You used indexing to combine the sentences before 'name' with the sentences after 'name'.
