Preparation

We first have to download smtplib and ssl module using PIP to use email service.

pip install smtpllib & pip install ssl First, we have to connect our computer to SMTPD server which sends email.

Run the following code.

python -m smtpd -c DebuggingServer -n localhost:1025 We first have to let Python login to our google gmail account.

Go to Google Account setting and click on allow.

Make sure you are not using not underage account.
