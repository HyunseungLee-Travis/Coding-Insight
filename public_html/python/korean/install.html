파이썬을 기기에 설치하기

파이썬이 이진수 (0과 1)로 컴파일이 되어서 기계가 이해하기를 도와주기 위해서 먼저 공식 사이트에서 컴파일러를 설치해야 합니다.











파이썬 설치 & 사용 방법
Computer Science Lab (CSL) 2






설치

1. 파이썬 사이트 설치 페이지로 들어가기.

2. 노란색 버튼 'Download Python 3.10.0' 누르기

3. 새로운 설치 박스가 브라우저 밑에 생기는데 이를 클릭하기



4. Add Python to Path 옵션을 클릭

5. Customize installation 버튼을 누르기



6. 모두 선택하고 Next 버튼을 누르기



7. 1부터 5번째 옵션을 선택하기

8. Install 버튼 누르기



9. Close 버튼을 눌러주세요.

사용 방법

저희는 파이썬의 IDLE이라는 편집기를 사용하겠습니다.

먼저 파이썬의 IDLE을 이용하려면 앱을 찾고 실행을 해야 합니다.

windows+s버튼을 누른 후 'idle'라고 치면 파이썬 IDLE가 뜹니다. 열기 또는 엔터를 누르세요.



파이썬 코드를 실행하기 위해서 >>> 다음에 넣고 엔터를 누르면 됩니다.



print("Hello World")라고 친 후 엔터를 누르겠습니다.
