ASCII란?

ASCII는 한국어로 '아스키'라고 발음합니다.

아스키는 American Standard Code for Information Interchange의 약자입니다. 한국어로는 "정보 교환을 위한 미국 표준 암호"를 뜻합니다.

ASCII는 알파벳의 모든 캐릭터를 0에서 127(자연수)로 나타낸 코드입니다. 예를 들면, 아무것도 없는 ' '는 0입니다.

퀴즈! 알파벳을 ASCII로 나타낼 때 a가 클까요? 아니면 A가 클까요? A라고 하셨나요? 흠… A가 크게 보이지만 ASCII에서는 a가 더 큽니다.
