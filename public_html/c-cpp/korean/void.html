void 데이터 형태

void 데이터 형태란 아무것도 돌려주지 않는다를 뜻합니다.

return을 사용해도 되지만 값을 돌려주면 안됩니다.

또한, 포인터를 사용할 때 값의 종류를 모른다면 void 형식을 사용할 수 있습니다.

void main() {} 형태

지금까지 int 데이터 형태를 이용해서 main 함수를 선언했습니다.

그리고 항상 return 0; 코드를 마지막에 넣었습니다.

int main() {
  return 0;
} 하지만 이제는 귀찮게 return 0;을 사용하지 않고 void 데이터 형태를 이용하면 return하지 않아도 됩니다.

void main() {

} 또는 void를 사용해 아무것도 돌려주지 않는 함수를 만들 수도 있습니다.

void testing() {
  printf("Hello World");
}
