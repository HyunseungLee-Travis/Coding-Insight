무한 반복문

무한 반복이란 사용자가 억지로 끝내기 전까지 무한으로 실행이 되는 반복문을 뜻합니다.

무한 반복문을 원하지 않을 때가 대부분이지만 원하는 때도 있을 수 있기 때문에 배우겠습니다.







무한 반복문
Computer Science Lab (CSL) 22






무한 반복문 만들기

무한 반복문을 만들기 위해서 while 반복문 조건을 넣는 곳에 1 (true)를 입력하면 됩니다.

조건문을 이용하여 break하는 것도 가능합니다. break에 관한 내용은 다음 시간에 배우겠습니다.



예시

#include <stdio.h>

int main()
{
    int a;

    while (1) {
      scanf("%d", &a);
      printf("%d", a);
    }

    return 0;
}
