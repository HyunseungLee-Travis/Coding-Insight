ctype.h 라이브러리

ctype 라이브러리는 어떤 문자가 숫자로만, 문자로만 등 확인하는 함수를 가지고 있는 라이브러리 이름입니다.

쉽게 설명하지면, ctype 라이브러리를 사용하여 C 언어에 있는 다양한 데이터 형태를 다룰 수 있습니다.

#include <ctype.h>

islower()와 tolower()

islower()이라는 함수는 문자를 입력하면 영문자가 소문자인지 확인하는 함수입니다.

if (islower('a')) {
    printf("lower");
} else {
    printf("not lower");
} tolower()은 문자를 소문자로 바꾸는 함수입니다.

if (islower('a')) {
    printf("lower");
} else {
    printf("not lower");
    printf("%c", tolower('a'));
}



isupper()와 toupper()

isupper()이라는 함수는 문자를 입력하면 영문자가 대문자인지 확인하는 함수입니다.

if (isupper('A')) {
    printf("upper");
} else {
    printf("not upper");
} toupper()은 문자를 대문자로 바꾸는 함수입니다.

if (islower('A')) {
    printf("upper");
} else {
    printf("not upper");
    printf("%c", toupper('A'));
}



프린트를 할 수 있는지

ctype.h의 isprint() 함수를 사용하면 그 문자가 터미널에 출력이 될 수 있는지 확인할 수 있습니다.

if (isprint('a')) {
    printf("printable!");
} else {
    printf("not printable :(");
}
