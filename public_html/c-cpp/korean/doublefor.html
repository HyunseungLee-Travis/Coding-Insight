이중 반복문

이중 반복문은 반복문이 두 개 중 하나가 다른 반복문 안에 있는 형태를 뜻합니다.







이중 반복문
Computer Science Lab (CSL) 20






예시 - 구구단 출력하기

이중 반복문은 규칙적으로 반복이 되는 변수가 2개 이상이면 사용되는 형태로 반복문 하나로는 불가능합니다.

2단부터 9단까지 구구단 프로그램을 작성하겠습니다.

#include <stdio.h>

int main()
{
    for (int i = 0; i < 9; i++) {
      for (int j=0; j < 9; j++) {
        printf("%d x %d=%d", i, j, i * j);
      }
    }

    return 0;
}
