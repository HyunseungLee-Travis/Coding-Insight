C++과 C 언어 중요한 차이

1. C 언어에 새로운 기능을 추가합니다 (객체 기능 - class)
2. C 언어는 키워드 32개를 가지고 있지만 C++은 63개입니다.
3. C는 malloc, calloc, free를 사용하지만 C++은 new, delete을 사용합니다.

C 언어보다 더 재미있을 것 같습니다!
