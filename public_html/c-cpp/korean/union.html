union과 struct

struct는 데이터의 모음입니다.

struct는 메모리 안에 나누어서 저장합니다.

하지만 union는 메모리 안에 모두 저장합니다.

union 선언

union DATA {
  int a;
  char aa;
} struct와 형태, '.' 사용은 다 같습니다.

참고로 struct와 같이 typedef와 같이 사용할 수 있습니다.

typedef union Person
{
    char namPer[99];
    int age;
    char gender; //M - 남자, W - 여자, E - 기타
} PersonStruct;

PersonStruct data;
data.age = 12;
