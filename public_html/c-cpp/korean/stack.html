stack이란?

stack은 쌓다를 뜻합니다.

stack은 데이터 종류의 하나로 값들을 쌓아 올리는 특징을 가지고 있습니다.

stack은 stack이라는 라이브러리에서 가지고 와야 합니다.

#include <stack>

stack 사용하기

stack은 stack<데이터형태> 형태로 선언하면 됩니다.

데이터를 추가하기 위해서 push()를 사용하세요.

#include <iostream>
#include <stack>
using namespace std;

int main()
{
  stack s;

  s.push(3);
  s.push(4);
  s.push(5);

  return 0;
}



pop()와 push()

pop()를 사용하면 stack의 마지막 아이템을 제거합니다.

push()를 사용하면 뒤에 새로운 아이템을 더합니다.

#include <iostream>
#include <stack>
using namespace std;

int main()
{
  stack<int> s;

  s.push(3);
  s.pop();

  return 0;
}



size()

size()를 사용하면 정수로 stack 안에 있는 아이템의 개수를 알 수 있습니다.

#include <iostream>
#include <stack>
using namespace std;

int main()
{
  stack<int> s;

  s.push(3);
  s.pop();

  cout << s.size();

  return 0;
}



top()

top()을 사용하면 stack에서 가장 위에 있는 아이템을 찾을 수 있습니다.

#include <iostream>
#include <stack>
using namespace std;

int main()
{
  stack<int> s;

  s.push(3);

  cout << s.top();
  s.pop();
  cout << s.top();

  return 0;
}`
