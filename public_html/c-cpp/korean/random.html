rand()

rand() 함수는 무작위로 0부터 2147483647(int의 가장 큰 값) 사이의 수를 선택합니다.

#include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main()
{
    srand(time(0));
    printf("%d", rand());

    return 0;
}
