파이썬을 기기에 설치하기

파이썬이 이진수 (0과 1)로 컴파일이 되어서 기계가 이해하를 도와주기 위해서 먼저 공식 사이트에서 컴파일러를 다운로드해야 합니다.



파이썬 설치 & 사용 방법 - Computer Science Lab (CSL) 2





설치

1. 먼저 파이썬 공식 사이트 다운로드 페이지로 들어갑니다.

2. 사이트를 들어가면 노란색으로 '3.9.6' 버튼이 있을 것입니다. 버튼을 클릭하고 기다리세요.

3. 조금 뒤에 새로운 다운로드 박스가 브라우저 밑에 생깁니다. 그 박스에 있는 퍼센트가 100%로 되면 클릭하세요.

4. 다음에, 파이썬 다운로드 마법사 창이 뜹니다. INSTALL 버튼을 누르세요.

5. 다양한 옵션이 있습니다. 모두 다 선택한 후 Next 버튼을 누르세요.

6. 이제는 파이썬 환경 설정 창이 뜹니다. 위에서 3, 4, 5 번쨰 옵션 (create shortcut, add Python environment variable, precompiled standard library). Install 버튼을 누르세요.

7. 몇 분 뒤에 파이썬 설치가 완료 될 것입니다.









사용 방법

저희는 파이썬의 IDLE이라는 편집기를 사용하겠습니다.


먼저 파이썬의 IDLE을 이용하려면 앱을 찾고 실행을 해야 합니다. windows+s버튼을 누른 후 'idle'라고 치면 파이썬 IDLE가 뜹니다. 열기 또는 엔터를 누르세요.

파이썬 코드를 실행하기 위해서 >>> 다음에 넣고 엔터를 누르면 됩니다. print("Hello World")라고 친 후 엔터를 누르겠습니다.
