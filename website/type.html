코드 def wowAnd(nextToWow): return "wow " + nextToWow에서 사용자가 실수로 nextToWow 매개변수의 값을 숫자로 입력할 수 있습니다.

이를 방지하기 위하여 다음과 같은 방법이 있습니다.





어떤 함수가 실행할 수 있는지의 여부를 알기
