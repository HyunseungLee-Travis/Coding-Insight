stack이란?

stack은 쌓다를 뜻합니다.

stack은 데이터 종류의 하나로 값들을 쌓아 올리는 특징을 가지고 있습니다.

stack은 stack이라는 라이브러리에서 가지고 와야 합니다.

#include <stack>

stack 사용하기

stack은 stack<데이터형태> 형태로 선언하면 됩니다.

데이터를 추가하기 위해서 push()를 사용하세요.

#include <iostream>
#include <stack>
using namespace std;

int main()
{
  stack s;

  s.push(3);
  s.push(4);
  s.push(5);

  return 0;
}
