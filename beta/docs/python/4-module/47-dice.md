# 주사위

주사위를 무작위로 굴려 출력하는 프로그램을 만들어 보겠습니다.

## 코드 설명

`⚀`, `⚁`, `⚂`, `⚃`, `⚄`, `⚅` 중 하나를 무작위로 선택해서 출력하겠습니다.

```py
import random
random.choice(["⚀", "⚁", "⚂", "⚃", "⚄", "⚅"])
```
